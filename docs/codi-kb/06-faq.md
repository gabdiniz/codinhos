# Perguntas frequentes

**Para que idade é o Codinhos?**
Para crianças e adolescentes de 11 a 14 anos. Todo o conteúdo, a linguagem e os desafios são pensados para essa faixa.

**Que linguagem de programação o aluno aprende?**
JavaScript, uma das linguagens mais usadas no mundo e uma excelente porta de entrada para programar.

**Precisa instalar alguma coisa?**
Não. O aluno escreve e roda o código direto no navegador. Basta acesso à internet.

**O aluno precisa já saber programar?**
Não. As trilhas começam do zero, pelos fundamentos, e evoluem de forma gradual.

**Quem monta o conteúdo das aulas?**
O conteúdo vem de um catálogo pronto e de qualidade, mantido pela equipe do Codinhos. O gestor da escola seleciona e ordena os módulos — não precisa criar nada do zero.

**Como funciona a avaliação dos desafios?**
A avaliação olha se o código funciona (o comportamento), o que permite várias soluções válidas. A escola escolhe entre avaliação automática, automática com revisão do educador, ou totalmente manual.

**O que é o Codi?**
É o tutor de IA que acompanha o aluno. Ele ajuda com dicas quando o aluno trava, mas nunca entrega a resposta pronta — o objetivo é fazer o aluno pensar e aprender.

**Como a escola acompanha o progresso?**
Pelo painel do gestor, com relatórios de quem completou o quê, por aluno e por turma.

**Dá para personalizar com a identidade da escola?**
Sim. Cada escola tem seu ambiente com cores e logo próprias.

**Como os alunos são cadastrados?**
Um a um ou importados por planilha (CSV), com um modelo pronto para baixar. Cada aluno recebe um convite por e-mail para o primeiro acesso.

**É seguro para crianças?**
Sim. Os dados de cada escola ficam separados, o Codinhos segue a LGPD e o tutor de IA tem proteções para manter as conversas adequadas à idade e no tema do aprendizado.

**Quanto custa?**
Os planos e condições são tratados diretamente com a nossa equipe, de acordo com a realidade de cada escola. Fale com a gente pelo formulário de contato para receber uma proposta.

**Como faço para levar o Codinhos para a minha escola?**
É só entrar em contato pelo formulário do site. A nossa equipe retorna para entender sua necessidade e apresentar o Codinhos em detalhe.
